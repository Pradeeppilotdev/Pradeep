# Portfolio — Chandrapradeep R

Personal portfolio built with Next.js, Three.js, GSAP, Lenis, and Tailwind CSS.

## Stack

- **Framework** — Next.js 16 (App Router)
- **3D** — React Three Fiber, @react-three/postprocessing, Three.js
- **Animations** — GSAP, ScrollTrigger, Framer Motion
- **Scrolling** — Lenis (smooth scroll)
- **Styling** — Tailwind CSS, CSS custom properties (dark/light theme)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
